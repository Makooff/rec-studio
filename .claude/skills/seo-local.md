# SEO Local — Nova

## Priorités SEO local

1. Google Business Profile (GBP) — impact immédiat Maps + recherche locale
2. Cohérence NAP (Nom, Adresse, Téléphone) sur toutes les plateformes
3. Mots-clés géo sur le site (balises + contenu)
4. Citations locales (annuaires)
5. Avis Google (quantité + réponses)

## Google Business Profile — setup complet

### Informations à compléter (100%)
- Nom exact de l'entreprise
- Catégorie principale + catégories secondaires
- Adresse complète (ou zone de service si nomade)
- Téléphone local (pas 0800)
- Horaires d'ouverture (+ jours fériés belges)
- Site web
- Description (750 chars max — intégrer 2–3 mots-clés géo)
- Photos : façade, intérieur, équipe, produits/services (min 10)
- Posts Google : 1/semaine minimum

### Attributs importants
- Accessible PMR
- Parking disponible
- Modes de paiement
- Langues parlées (FR / NL pour Belgique)

## Mots-clés géo — stratégie

**Format cible :** `[service] [ville/région]`

Exemples :
- "agence vidéo Bruxelles"
- "production vidéo publicitaire Belgique"
- "création site web Bruxelles"
- "référencement local Bruxelles"
- "campagne Meta Ads Belgique"

**Outils de recherche :**
- Google Keyword Planner (volumes BE + FR)
- Google Search Console (positions actuelles)
- Ubersuggest / Semrush (concurrence)

## Structure pages SEO local

### Page d'accueil
- H1 : `[Service principal] à [Ville] | [Nom entreprise]`
- Contenu : mentionner la zone géo naturellement (3–5x)
- Schéma LocalBusiness en JSON-LD

### Pages de service géolocalisées (si multi-ville)
```
/agence-video-bruxelles
/agence-video-liege
/agence-video-ghent
```
Contenu unique par page (pas de duplicate content).

### Page contact
- Adresse complète
- Carte Google Maps intégrée
- Schéma Organization en JSON-LD

## Citations locales — Belgique

Annuaires prioritaires à créer/revendiquer :
1. Google Business Profile
2. Bing Places
3. Facebook / Instagram (adresse)
4. LinkedIn (page entreprise)
5. Gouden Gids / Pages d'Or (goldenguide.be)
6. Chambre de Commerce Bruxelles
7. 1819.be (portail entreprises bruxellois)
8. TrustPilot
9. Yelp

Cohérence NAP stricte sur TOUS les annuaires.

## Gestion des avis Google

**Objectif :** 20+ avis, note > 4.5

Stratégie de collecte :
- Lien direct avis Google : `https://g.page/[votre-id]/review`
- Envoyer par SMS/email après chaque projet
- QR code sur devis / facture

Répondre à TOUS les avis (positifs + négatifs) sous 48h.
